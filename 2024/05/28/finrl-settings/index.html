补充：

  1. 激活你需要作为kernel的虚拟环境
  2. 安装ipykernel
    conda install ipykernel
  3. 添加kernel
    python -m ipykernel install –user –name [虚拟环境名称]
  4. 查看所有kernel
    jupyter kernelspec list
  5. 删除kernel
    jupyter kernelspec remove [kernel名称]
